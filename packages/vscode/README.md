# stilla / vscode

> `Still` :: A quiet, introverted theme that stays out of your way.

VS Code color theme for Stilla.

## Development

Theme colors are configured in `src/colors.ts`. `src/theme.ts` contains the VS Code configuration using the color palette, and `src/build.ts` generates the final theme JSON.

Do not edit anything in `themes/` manually -- it is autogenerated. Instead, modify `src/colors.ts` or `src/theme.ts`, then run:

```sh
bun run build
```

To package a `.vsix`:

```sh
bun run package
```
