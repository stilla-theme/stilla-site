# stilla / emacs

> `Still` :: A quiet, introverted theme that stays out of your way.

Doom Emacs theme for Stilla. Requires [doom-themes](https://github.com/doomemacs/themes).

## Usage

Copy `doom-stilla-theme.el` into your doom-themes themes directory, then:

```elisp
(setq doom-theme 'doom-stilla)
```
