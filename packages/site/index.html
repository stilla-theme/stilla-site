Install

VS Code

Search Stilla in Extensions, or:

code --install-extension jakeisnt.stilla

Neovim

Add to lazy.nvim:

{
  "stilla-theme/stilla-site",
  config = function()
    vim.cmd.colorscheme("stilla")
  end,
}

Doom Emacs

Download doom-stilla-theme.el, then:

(setq doom-theme 'doom-stilla)
